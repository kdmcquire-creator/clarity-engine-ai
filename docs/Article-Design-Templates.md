# Clarity Engine: 10 Visually Appealing Article Design Templates

These are **visual layout templates** with distinct design styles, typography approaches, imagery strategies, and interactive elements. Choose the ones that resonate with your brand voice.

---

## Template 1: "Hero + Sidebar" (Modern Professional)

**Visual Style:** Clean, minimalist with accent colors  
**Layout:** Large hero image (70% width) + sidebar with key takeaways (30% width)  
**Typography:** Bold sans-serif headlines, elegant body copy  
**Color Scheme:** Navy blue + coral accents on white background  
**Interactive Elements:** Sticky sidebar with progress bar, collapsible sections  
**Best For:** Technical guides, comparison articles, how-to content  

```
┌─────────────────────────────────────┐
│ HERO IMAGE (70%)    │ KEY TAKEAWAYS │
│                     │ • Point 1     │
│                     │ • Point 2     │
│                     │ • Point 3     │
├─────────────────────┼───────────────┤
│ ARTICLE CONTENT     │ SIDEBAR CTA   │
│ (Full width below)  │ (Sticky)      │
└─────────────────────────────────────┘
```

## Template 2: "Timeline Vertical" (Journey-Focused)

**Visual Style:** Narrative, chronological flow  
**Layout:** Vertical timeline with alternating left/right content blocks  
**Typography:** Serif headers, sans-serif body, large drop caps  
**Color Scheme:** Gradient background (blue to purple), white cards  
**Interactive Elements:** Animated timeline markers, hover effects on cards  
**Best For:** Evolution articles, trend analysis, step-by-step guides

## Template 4: "Infographic + Text" (Data-Driven)

**Visual Style:** Educational with custom illustrations  
**Layout:** Alternating full-width infographics + text sections

## Template 5: "Split Screen" (Comparison-Focused)

**Visual Style:** Balanced, symmetrical design  
**Layout:** Two-column layout with distinct visual treatment for each side  
**Typography:** Bold headers, contrasting fonts for left vs. right  
**Color Scheme:** Left side (blue), right side (coral), neutral background  
**Interactive Elements:** Toggle switch to flip views, hover highlights  
**Best For:** Comparisons, pros/cons, before/after  

```
┌──────────────────┬──────────────────┐
│   OPTION A       │   OPTION B       │
│   [Content]      │   [Content]      │
│   [Features]     │   [Features]     │
│   [Pricing]      │   [Pricing]      │
└──────────────────┴──────────────────┘
```

## Implementation Recommendations

**For Clarity Engine:**

1. **Template 1 (Hero + Sidebar)** - Use for "Semrush vs Ahrefs" and tool comparisons
2. **Template 4 (Infographic + Text)** - Use for research-backed articles with statistics
3. **Template 5 (Split Screen)** - Use for all comparison articles
4. **Template 8 (Resource Hub)** - Use for guides with downloadable templates
5. **Template 10 (Data Dashboard)** - Use for performance reports and analytics articles

**Design System Consistency:**
- Use Clarity Engine brand colors (blue, coral) across all templates
- Maintain consistent typography (sans-serif headers, readable body)
- Add subtle animations (fade-ins, hover effects) for engagement
- Ensure mobile responsiveness for all layouts
- Include author bio cards at the end of each article
